# EchoNote 🪞

*EchoNote — LLM 搭載のシンプルな多言語日記 & ライティングアプリ。*

[English](./README.md) | [简体中文](./README.zhs.md) | [繁體中文](./README.zht.md) | 日本語

---

EchoNote は、**AI を活用した執筆と振り返り**にフォーカスした、シンプルで集中しやすいライティング体験を提供します。どの言語でも日記を書け、ローカルの Markdown ファイルとして保存したうえで、AI に要約・推敲・翻訳を依頼できます。

---

## 🤖 AI 機能

- **AI 日記アシスタント**
  - AI との対話・続きを書く形で 1 日分の日記をまとめて生成
  - リアルタイムのスペル・文法チェックと軽めのヒント
  - 行単位のスマート補完と、文レベルの意味的サジェスト
  - 執筆後に内容を分析し、改善提案を提示

- **多言語ライティング & 翻訳**
  - 複数言語での執筆に対応（日本語・中国語・英語などに限定されない）
  - AI の要約は、原文の言語とトーンをできるだけ維持
  - 翻訳機能と組み合わせることで、語学学習の相棒としても利用可能

- **AI 要約と振り返り**
  - 各日の日記から Emoji と短い要約を生成
  - タイムラインビューで最近の日々を素早く振り返り

## ✨ 機能一覧

### 📝 カレンダーエディタ

- **リッチな Markdown 対応**
  - リアルタイム構文ハイライトとインライン整形
  - 自動保存機能
  - 見出し・リスト・コードブロック・引用などをサポート

### 📅 カレンダー & タイムライン

- **カレンダー**
  - その日の Emoji と執筆状況を示すビジュアルカレンダー

- **タイムライン**
  - 直近の日記と AI 要約を時系列に一覧表示

### 🌐 多言語サポート

- 翻訳機能と組み合わせて、さまざまな言語での執筆を練習
- AI 要約は、可能な限り元のテキストと同じ言語・視点を維持

### 🖥️ クロスプラットフォーム

- **デスクトップ**：Windows / macOS / Linux
- **モバイル**：Android / iOS / iPadOS

## 🧠 複数の AI API プロバイダ

EchoNote は複数の OpenAI 互換 API に対応しており、用途に応じて AI を選択・無効化できます。

- [x] ChatGPT（OpenAI API）
  - [x] DeepSeek
- [x] 任意の OpenAI 互換プロバイダ（設定画面で `baseUrl` を指定）
- [x] AI を使用しない（ローカル Markdown のみ）
- [ ] Claude（Anthropic）
- [ ] Azure OpenAI
- [ ] Gemini（Google）
- [ ] Cloudflare
- [ ] その他

> **設定** から利用するプロバイダを切り替えたり、互換サービスごとにカスタム設定を追加したりできます。

## 🧱 技術スタック

- **フロントエンド**：Bun + SvelteKit + Vite + CSS
- **バックエンド**：Tauri + Rust

## 🚀 開発

```bash
bun install
bun run dev          # Web 開発サーバーを起動
bun run tauri dev    # デスクトップアプリを起動（事前に `bun run build` が必要）
```

## 🎯 コンセプト

- **ミニマルな執筆体験**：シンプルな UI で執筆の邪魔をしない
- **AI はアシスタントとして**：控えめな提案と要約で創作をサポート
- **データのオーナーシップ**：すべてのエントリはプレーンな Markdown として保存され、簡単に閲覧・バックアップ可能
- **一貫したクロスプラットフォーム UX**：すべてのデスクトッププラットフォームで同じレイアウトと操作感を提供
