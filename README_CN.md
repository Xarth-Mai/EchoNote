# EchoNote 🪞

*EchoNote（回声记）— 一款由 LLM 驱动的简约智能双语日记与写作应用。*

[English](./README.md) | 中文

---

EchoNote 提供优雅、专注的写作体验。它将 AI 辅助的表达、反思和总结功能相结合，同时保持完整的离线能力。

---

## ✨ 功能特性

### 📝 智能 Markdown 编辑器（核心）

- **丰富的 Markdown 支持**
  - 实时语法高亮和内联格式化
  - 自动保存功能
  - 支持标题、列表、代码块、引用等

- **AI 辅助写作**
  - 实时拼写与语法错误检测，提供轻量级提示
  - Tab 键接受建议
  - 自动补全与语义提示（句子级别）
  - 选择文本时可选翻译提示

- **AI 总结与分析**
  - 写作完成后自动分析
  - 语法与逻辑修正（必要时）
  - 生成：写作摘要、情感分析、关键词提取
  - 保存到 front matter 的 `ai_summary` 字段

### 📅 心情日历与时间线视图

- **日历界面**
  - 显示每日心情和写作状态的视觉日历视图
  - 每个日期的心情表情标记
  - 点击任意日期打开/编辑对应条目
  - 手动编辑心情（与元数据同步）

- **时间线视图**
  - 显示近期条目及其摘要
  - 滚动时自动加载更早的内容
  - 显示 AI 摘要和条目预览
  - 响应式布局：
    - **竖屏**：日历 + 时间线（紧凑，可展开）
    - **横屏**：分屏视图（左侧日历，右侧编辑器，支持全屏）

### 🖥️ 跨平台支持

- **桌面端**：Windows, macOS, Linux
- **移动端**：Android, iOS, iPadOS

---

## 🧱 技术栈

- **前端**：HTML + TypeScript + Tailwind CSS
- **后端**：Tauri + Rust

## 🎯 核心原则

- **简约写作**：简洁界面，不干扰创作
- **AI 辅助**：非侵入式智能修正和总结
- **数据所有权**：所有日记均为 Markdown 文件，可直接打开和备份
- **一致的跨平台体验**：所有桌面平台使用相同的界面逻辑
